# Install
```
  git clone https://github.com/artas90/nvchad-config.git ~/.config/nvim && nvim
```

# Uninstall
```
  rm -rf ~/.config/nvim && \
  rm -rf ~/.local/share/nvim && \
  rm -rf ~/.cache/nvim
```

with sudo
```
  sudo rm -rf ~/.config/nvim && \
  sudo rm -rf ~/.local/share/nvim && \
  sudo rm -rf ~/.cache/nvim
```
